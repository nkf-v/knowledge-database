Code Injection - это уязвимость веб-безопасности, которая позволяет злоумышленнику вмешиваться в логику приложения, через параметры.

Такая уязвимость проявляется когда в запросе ожидается код на исполнение. Когда работа приложения плохо спроектирована и приходится вот так передавать логику приложения между слоями и выполнять ее на стороне.
