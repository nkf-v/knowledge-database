Code Injection - это уязвимость веб-безопасности, которая позволяет злоумышленнику вмешиваться в логику приложения, через параметры, и встраивать код в память приложения. Когда работа приложения плохо спроектирована и приходится вот так передавать логику приложения между слоями и выполнять ее на стороне.
